## Purpose
These screenshots are retained as **supporting evidence artifacts** for **Proof Pack 01**.

They demonstrate:
- baseline budget context
- actual spend accumulation
- service-level driver analysis
- forecasted month-end exposure

All screenshots are:
- captured from Azure Cost Management
- time-bound to the stated billing period
- referenced by Evidence ID (EV-YYYY-MM-XXX)
- linked from the Evidence Index

No screenshots imply production authority, engineering control, or production tenant ownership.
